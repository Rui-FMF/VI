Motivation

Studies estimate that 10,4% of the world GDP, 7.5 millions of millions of euros, "desapear" in offshores. Countries loose an considerate amount of money that could be used in providing better welfare. Investigative journalism does a very importante job in exposing some of this offshores, Panama Pappers, Bahama Leaks, etc. Using this data this project aims to create visualizations that show the number of offshores worldwide nowadays and through time.

Users

The target users for this visualizations would be users familiar with the concept of offshores. With the project's vizualizations the users could answer the following questions:

  • That we know of, how many companies resort to offshores around the world?
  • How many active offshores exist in a given Year? Month? Country?
  • Has the number of offshores been rising in my country?
  • Which are the most common offshore paradises that my country companies use?

Datasets

For this visualization project it was used the Offshore Leaks Database available at: https://offshoreleaks.icij.org/ .

This dataset, nodes-entities.csv file, contains data regarding offshore companies, foundations and trusts from the Pandora Papers, Paradise Papers, Bahamas Leaks, Panama Papers and Offshore Leaks investigations.

Using this vast dataset 3 smaller ones were generated in order to facilitate its usage in the vizualizations

  • Dataset1 - Used in the choropleth map visualization.
    • Consists on the number of active offshores per country per year.
  • Dataset2 - Used in the sankey visualization.
    • Consists on the number of active offshores per origin country per year per offshore country.
  • Dataset3 Used in the temporal visualization
    • This dataset contains the number of offshore companies at a given date (year and month) for each country.
